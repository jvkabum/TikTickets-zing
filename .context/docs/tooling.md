# Ferramentas e Padrões (Tooling)

## Frontend (Vue 3.5)
- **Framework**: [Quasar 2.17+](https://quasar.dev/)
- **Estetização**: CSS padrão e componentes Quasar.
- **Gestão de Estado**: Pinia.
- **Busca de Dados**: [Tanstack Vue Query](https://tanstack.com/query/latest/docs/framework/vue/overview).
- **Validação de Formulários**: Vee-validate + Zod.
- **Utilidades**: VueUse.
- **Testes**: Vitest.

## Backend (Node.js)
- **Runtime**: ts-node-dev para desenvolvimento.
- **ORM**: Sequelize com Sequelize-typescript.
- **Fila**: Bull (Redis).
- **Integração WhatsApp**: `whatsapp-web.js`.

## Qualidade de Código
- **Linting**: ESLint + Prettier.
- **Processo**: PREVC (P, R, E, V, C).
- **Commits**: Padronizados via habilidade `commit-message` em `.context/skills/`.
